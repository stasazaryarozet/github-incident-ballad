## Как читать

Баллада — история с уроком.  
Рассказанная ритмично.  
Структурированная актами.  
Завершённая извлечением.

Форма: короткие строки создают ритм.  
Один факт — одна строка.  
Одна мысль — одна строфа.  
Один акт — одна стадия.

Повествование: начало → кризис → действие → разрешение → урок.  
Драматургия: проблема обнаружена → причина установлена → решение найдено → знание извлечено.

Объективность: каждый факт проверяем.  
Временные метки точны.  
Цитаты прямые.  
Источники указаны.

Балладность: не стихи, но ритм.  
Не поэзия, но музыка фактов.  
Повествование через события.  
Урок через опыт.

Учит сама собой:  
Читаешь — чувствуешь ритм.  
Следуешь структуре — понимаешь метод.  
Видишь форму — усваиваешь принцип.

Если можешь написать подобное —
